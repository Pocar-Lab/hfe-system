Processed analysis outputs live here, organized by data log type.

- `data/processed/calibration/` holds thermocouple calibration tables.
- `data/processed/HX_performance/` holds HX-performance reference data.
- `data/processed/leak_test/` is where ORCA writes the processed leak-analysis series used to recreate plots.
- Raw logger files live in `data/raw/<type>/`.
