# HX Reconstructed-From-History Evidence

These files document the historical values and recovered notebook plots used to reconstruct the four missing Oct 23/24 HX raw-like logs.

Primary tables:
- `tables/historical_run_inventory.csv`: run names, durations, starts, and file status.
- `tables/historical_fit_targets.csv`: historical fit-window power, UA, heat flux, DeltaT, and slopes.
- `tables/historical_window_summaries.csv`: historical rolling-window medians and percentiles.
- `tables/historical_energy_targets.csv`: historical integrated HX energies.
- `tables/historical_heat_leak_target.csv`: historical warmup heat-leak result.
- `tables/historical_output_inventory.csv`: extracted original notebook plot and text-output inventory.

The PNGs under `original_plots/` are recovered notebook outputs. The corresponding reconstructed raw-like CSVs are still marked as reconstructed and should not be treated as original logger exports.
